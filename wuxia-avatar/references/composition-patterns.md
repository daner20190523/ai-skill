# 章法谱 — 构图九式

选一式即可，不混用。

## 九式概要

### 一式·行云流水（流程）
画：来龙（左/上）→ 中间小颠操作 → 去脉（右/下），朱砂或金线走主流向。适合输入→处理→输出、自动化链路。

### 二式·管中窥豹（系统局部）
画：只取 3-5 个核心模块，小颠卡在其中一个里干活。适合系统局部、信息流、过滤。

### 三式·双峰并峙（前后对比）
画：左峰乱，右峰序；左旧，右新；左慢，右快。中间的过渡线是金或朱砂。适合新旧对比、A vs B、有/无对比。

### 四式·七情图谱（角色状态变）
画：2-4 个小墨小颠，每个一个状态，每个配一个短批注。适合心境变化、痛点演进、卡住到跑起来。

### 五式·借物喻理（概念隐喻）
画：一个大的「理器」（老物件或怪装置），少量入，一口出。要有记忆点。每次重新造喻。适合抽象概念具象化。

### 六式·九层之台（分层框架）
画：一层层宣纸/木板/石阶叠起来，不用金字塔。小颠在上面的一层铺砖或在下层打地基。适合框架、层级、能力栈。

### 七式·曲径通幽（地图路线）
画：一条蜿蜒小径，3-4 个路标节点。小颠站在起点指路、或在中途举葫芦休息、或在终点等你。适合成长路径、学习路线、承接链路。

### 八式·浮生三帧（小漫画分镜）
画：2-4 格，每格只有一个动作。适合吐槽、变化过程、失败到成功、段子。

### 九式·一语点穴（核心洞察/金句）
画：极简，一个视觉焦点（老物件+一缕光/一滴墨），小颠在一旁抱胸点头或倚着物件举杯。留白≥ 60%。批注不超过 2 字。适合金句、穿透性结论。

## 造喻三步法

每次从正文重新造喻，不翻旧稿：

1. **抽象变动作**——把"同步""收敛""过滤"换成物理动作：磨、筛、斟、折、晾、压、打捞
2. **系统变老物件**——把"数据流""状态机""层架构"换成江湖/日常老物件（见下谱）
3. **小颠上手干**——他不是在解释这个物件，他就是操作这个物件的人

## 江湖物件谱

首推（与墨小颠天然亲和）：

| 物件 | 喻什么 |
|------|--------|
| 老砚台 | 沉淀、积累、底蕴 |
| 旧杆秤 | 权衡、判断、取舍 |
| 竹简卷轴 | 知识、传承、记录 |
| 纸灯笼 | 线索、发现、照亮 |
| 老茶壶 | 泡出来、酿出来、时间的朋友 |
| 算盘 | 算计、效率、精确 |
| 折扇 | 展开、分层、开合 |
| 药柜（多抽屉） | 分类、索引、查找 |
| 染布缸 | 沾染、浸透、耳濡目染 |
| 线团 | 缠绕、解耦、理顺 |
| 石磨 | 研磨、转化、慢工出细活 |

也通（从日常中借用）：

| 物件 | 喻什么 |
|------|--------|
| 水井 | 深挖、源头 |
| 梯子 | 攀登、渐进 |
| 钥匙与锁 | 解锁、匹配 |
| 筛子 | 过滤、筛选 |
| 晾衣绳 | 晾晒、展示、并行 |
| 指南车 | 方向、导航 |
| 旧门环 | 门槛、入口 |

每次只选 1-2 个物件。物件是喻体，不是主体——墨小颠用它来表达意思才是主体。

## 动作谱

磨、斟、筛、晾、扛、拉、搭、铺、挂、钉、量、称、守、推、接、拆、卷、展、点、指、签字、画圈、勾、叉、点头

## 不复刻

每篇文章都从零造喻。以下旧构图绝不直接复用：
- 传送带断点
- 砚台磨出包浆
- 老茶壶越泡越香
- 算盘算效率
- 灯笼照路
- 墨小颠举葫芦看坑

同主题也要换花样。"12年经验"可以画砚台包浆，也可以画一棵老树年轮；可以画老茶壶越用越润，也可以画旧杆秤上刻满痕迹。

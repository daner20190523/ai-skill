# 快速参考用例

本文档展示常见排版需求的**原创解决方案**示例，可直接参考思路。

---

## 案例一：个人博客首页

### 需求
追求独特个性，不想像其他人那样用WordPress默认主题

### 方案："几何朋克"

**设计方向**：孟菲斯+极简混搭

**配色**：
- 背景：#FAFAFA（微灰白）
- 主色：#FF6B6B（珊瑚红）
- 强调：#4ECDC4（湖蓝）
- 文字：#2C3E50（深灰）

**布局核心**：
1. 顶部：超大标题 + 几何装饰
2. 左侧：头像融入不规则图形
3. 右侧/下方：文章列表用卡片+标签

**原创点**：
- 标题文字用渐变色
- 每篇文章卡片有不同的几何角标
- 鼠标悬停时卡片"弹跳"效果

## 案例二：产品介绍页

### 需求
想让产品看起来更高端、更有价值感

### 方案："剧院灯光"

**设计方向**：新锐极简+微交互

**配色**：
- 背景：纯黑 #0A0A0A
- 主文字：纯白 #FFFFFF
- 强调色：金色 #D4AF37 或 宝石蓝 #0066CC

**布局核心**：
1. 首屏：大字号Slogan + 巨大产品图
2. 特点区：数字+简短描述，滚动时数字递增
3. 细节区：大图轮播 + 详细参数
4. CTA：底部醒目行动按钮

**原创点**：
- 背景微妙的噪点纹理
- "灯光扫过"的入场动画
- 图片有电影宽银幕感

## 案例三：活动/事件页面

### 需求
营造紧迫感、让人想立即行动

### 方案："波普警报"

**设计方向**：波普艺术+故障艺术

**配色**：
- 警戒红：#FF0033
- 电光黄：#FFE600
- 赛博蓝：#00F0FF
- 纯黑：#000000

**布局核心**：
1. 倒计时：超大数字，秒数跳动
2. 信息块：用对话框样式
3. 福利区：用标签卡片叠加

**原创点**：
- 倒计时数字有"故障"效果
- 重要信息用"警告"样式
- 按钮像漫画的喊话泡

## 案例四：关于/个人简介页

### 需求
让人快速记住我是谁

### 方案："自画像"

**设计方向**：孟菲斯+个人元素

**配色**：
- 提取个人最爱颜色作为主色
- 辅以对比色

**布局核心**：
1. 名字超大居中
2. 一句话介绍
3. 关键标签（技能/标签/城市）
4. 社交链接

**原创点**：
- 名字做成类似签名的艺术字
- 标签用不同几何形状包装
- 个人特色元素（比如画自己的卡通形象）

## 案例五：作品集/案例展示

### 需求
让作品看起来更专业、更有吸引力

### 方案："画廊"

**设计方向**：极简+空间感

**配色**：
- 纯白或浅灰背景
- 单色强调

**布局核心**：
1. 栅格布局展示作品缩略
2. 点击放大modal查看详情
3. 作品标题叠加在图片上

**原创点**：
- 鼠标悬停时图片轻微放大+遮罩变化
- 加载时有"逐张出现"动画
- 空状态有创意的插画

## 案例六：404错误页

### 需求
让404也很有趣，减少用户流失

### 方案："小故障"

**设计方向**：故障艺术+波普

**配色**：任意高对比

**布局核心**：
1. 巨大"404"数字
2. 简短文案
3. 引导按钮

**原创点**：
- 404数字有Glitch效果
- 可以做个简单的404动画
- 按钮文字有趣（比如"带我回家"）

## 通用技巧速查

### 让文字更有冲击力

1. 减少文字量 → 提取核心关键词
2. 字号大 → 至少是周围的2倍
3. 字重粗 → 用900-weight字体
4. 颜色跳 → 用主题强调色
5. 加装饰 → 文字下加线条/图形

### 让布局更有设计感

1. 不对称 → 打破均等分割
2. 叠压 → 元素相互覆盖
3. 留白 → 给呼吸空间
4. 线条 → 用线条引导视线
5. 重复 → 统一元素反复出现

### 让人想点击（CTA）

1. 颜色对比强
2. 位置显眼（首屏/底部）
3. 动效提示（微弱脉冲）
4. 文字有行动感（"立即"/"开始"）
5. 周围有空白

## 常见排版雷区

### 避免
- ❌ 满屏都是重点 = 没有重点
- ❌ 颜色太多 = 杂乱
- ❌ 字体太多 = 不专业
- ❌ 元素紧贴 = 紧迫感
- ❌ 没有层次 = 阅读累

### 推荐
- ✅ 主次分明
- ✅ 配色克制（3-4色）
- ✅ 字体的系统（2种以内）
- ✅ 有呼吸感
- ✅ 视觉层级清晰
